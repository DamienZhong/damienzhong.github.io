学习Linux需要一个环境，我们需要创建一个虚拟机，然后在虚拟机上安装一个Centos系统来学习

  1. 先安装virtual machine ,vm12
  2. 再安装Linux (CentOS 6.8)
  3. 原理示意图（VM和CentOS的关系）

    安装VMware

  4. 去BIOS里修改设置开启虚拟化设备支持（f2, f10）
