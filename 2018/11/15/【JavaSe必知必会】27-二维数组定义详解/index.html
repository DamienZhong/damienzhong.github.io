二维数组概述

二维数组其实就是一个元素为一维数组的数组。

二维数组定义格式

格式1

  • 数据类型[][] 变量名 = new 数据类型[m][n];
  • m表示这个二维数组有多少个一维数组
  • n表示每一个一维数组的元素个数
  • 举例：
    • int[][] arr = new int[3][2];
    • 定义了一个二维数组arr
    • 这个二维数组有3个一维数组，名称是arr[0],arr[1],arr[2]
    • 每个一维数组有2个元素，可以通过arr[m][n]来获取
    • 表示获取第m+1个一维数组的第n+1个元素

格式2

数据类型[][] 变量名 = new 数据类型[m][];

  • m表示这个二维数组有多少个一维数组
  • 这一次没有直接给出一维数组的元素个数，可以动态的给出。
  • 举例：
    • int[][] arr = new int[3][];
    • arr[0] = new int[2];
    • arr[1] = new int[3]
    • arr[2] = new int[1];
